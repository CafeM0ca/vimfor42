# vimfor42
vim setting all!

## run
> git clone https://github.com/JinyRubRub/vimfor42.git
> cd vimfor42
> curl -fLo ~/.vim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
> chmod 777 ./run.sh
> ./run.sh


## open the vim
open your vim
> vim

write this command at command mode
> :PlugInstall
